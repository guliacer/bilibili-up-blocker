# B站UP主快速拉黑（无确认终极版）

一个面向 Bilibili 网页端的 Tampermonkey 油猴脚本。

当鼠标悬停在 UP 主名称区域时，页面会自动显示“拉黑该UP”按钮，点击后直接调用 B 站接口完成拉黑操作，无需再手动进入个人空间或经过额外确认弹窗，适合需要快速清理内容流的使用场景。

## 功能介绍

- 鼠标悬停 UP 主名称时自动显示快捷拉黑按钮
- 点击按钮后直接调用 B 站拉黑接口
- 支持动态页面中的常见 UP 主名称区域识别
- 按钮会随滚动和窗口变化自动更新位置
- 内置加载态、超时处理、网络异常处理和重复点击保护
- 操作成功或失败时会显示页面提示消息

## 适用环境

- 浏览器：Chrome、Edge、Firefox 等支持 Tampermonkey 的浏览器
- 脚本管理器：Tampermonkey
- 适用站点：`*.bilibili.com`
- 使用前提：需要先登录 Bilibili 账号

## 安装方法

### 方法一：本地导入

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)。
2. 打开 Tampermonkey 管理面板。
3. 新建脚本。
4. 将 [B站UP主快速拉黑.js](./B%E7%AB%99UP%E4%B8%BB%E5%BF%AB%E9%80%9F%E6%8B%89%E9%BB%91.js) 的全部内容复制进去并保存。
5. 打开或刷新 Bilibili 页面开始使用。

### 方法二：GitHub 仓库安装

如果你将此项目上传到 GitHub，可以在仓库中直接打开脚本文件，配合用户脚本安装链接或 Raw 地址进行安装。

## 使用方法

1. 登录 Bilibili。
2. 打开首页、视频页、动态页或其他能看到 UP 主名称的页面。
3. 将鼠标移动到 UP 主名称或其链接区域。
4. 页面会出现“拉黑该UP”按钮。
5. 点击按钮后，脚本会直接执行拉黑操作。
6. 页面右上角会弹出提示，告知操作成功或失败。

## 脚本特点

- 不依赖手动确认弹窗，操作路径更短
- 对动态渲染页面更稳定，减少按钮闪烁或异常消失
- 使用单按钮复用机制，避免重复创建节点
- 请求失败后会恢复按钮状态，便于再次尝试

## 注意事项

- 本脚本会直接执行拉黑操作，请谨慎点击。
- 必须在已登录 Bilibili 的状态下使用，否则无法获取接口所需凭证。
- 如果 B 站页面结构发生较大变化，部分页面的 UP 主识别可能需要后续更新。
- 脚本依赖 B 站当前接口行为，如官方接口调整，功能可能受到影响。

## 常见问题

### 1. 按钮没有出现

可能原因：

- 当前页面的 UP 主名称结构未被脚本命中
- 页面还未完全加载完成
- Tampermonkey 未启用脚本

建议：

- 刷新页面后重试
- 确认脚本已启用
- 打开浏览器控制台查看是否输出脚本加载日志

### 2. 点击后拉黑失败

可能原因：

- 未登录 Bilibili
- 网络请求超时或被浏览器环境拦截
- B 站接口返回失败信息

建议：

- 确认账号已登录
- 稍后重试
- 检查 Tampermonkey 权限是否正常
